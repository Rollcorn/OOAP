В  java поддержка концепции "класс как модуль" осуществляется слежующим образом:
1. Наименьшей единицей трансляции является файл .java в java все модули являются классами
или интерфейсам.
2. Классы распологаются в модулях - package. package является областью определенной областью
видимости для функций классов располаающихся в том же пэкедже. Функции из одного класса могут
быть доступны в другом классе при отсутсвтвии явного указания модификатора видимости public, privat, protected
3. Пэкэджи могут быьт объединены в библиотеки - модули предоставляющие функционал определенной функциональной
целью
4. Наиболее популярные и чаще всего необходимые библиотеки объединены в фрейморк Spring.
5. Библиотеки могут быть интегрированны в программу средствами maven/gradle в которых с помощью xml файла
конфигурируются зависимости на определенные библиотеки или фреймворки. maven/gradle хранят в своих репозиториях
версии библиотек что позволяет легко переключатся между ними при необходимости.
